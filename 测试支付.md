## 🧪 测试步骤

### 1. 启动开发服务器

```bash
npm run dev
```

### 2. 访问结账页面

打开浏览器访问：**http://localhost:3000/checkout**

### 3. 填写物流信息

在结账页面填写：
- 收货人姓名
- 联系电话
- 邮箱地址
- 详细地址
- 城市、省/州、邮政编码
- 国家/地区

### 4. 点击"继续支付"

系统会跳转到 Stripe Checkout 页面

### 5. 使用测试卡号支付

在 Stripe Checkout 页面，使用以下测试卡号：

**成功支付测试卡**：
- **卡号**: `4242 4242 4242 4242`
- **过期日期**: `12/34`（任意未来日期）
- **CVC**: `123`（任意 3 位数字）
- **邮编**: `12345`（任意 5 位数字）

### 6. 完成支付

支付成功后，您会被重定向到成功页面，显示：
- ✅ 支付金额
- ✅ 物流信息
- ✅ 订单确认
